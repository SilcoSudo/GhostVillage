# VI. QUY ЖҜб»ҡC QUбәўN LГқ (Cб»°C QUAN TRб»ҢNG)

 
1. KhГҙng trб»ҷn vai trГІ

| Layer           | ДҗЖ°б»Јc lГ m    | KhГҙng Д‘Ж°б»Јc lГ m  |
| --------------- | ----------- |---------------  |
| Client          | UI, gб»Қi API | Logic nghiб»Үp vб»Ҙ |
| Backend Service | Nghiб»Үp vб»Ҙ   | UI message      |
| DataAccess      | Query DB    | If/else         |
| DB              | LЖ°u data    | Logic           |

 

---
 2. TГӘn gб»Қi thб»‘ng nhбәҘt - CamelCase

* `*Controller.js` вҶ’ nhбәӯn request
* `*Service.js` вҶ’ nghiб»Үp vб»Ҙ
* `*DataAccess.js` вҶ’ MongoDB
* `*Model.js` вҶ’ schema
* `*Routes.js` вҶ’ route definitions
* `*Middleware.js` вҶ’ middleware functions

 
 Д‘Гўy so vб»ӣi cбәҘu trГәc nГ y
